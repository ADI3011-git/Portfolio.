# Aditya Khairwad — Portfolio

Premium personal brand site built around three pillars: Supply Chain, AI in Supply Chain, and Kinaxis.

## Structure
- `index.html` — markup for all sections
- `css/style.css` — design system + styles
- `js/script.js` — network graph visualization, nav, animations
- `assets/images/` — reserved for custom images (none required by default)

## Run locally
Just open `index.html` in a browser, or serve the folder:
```
python3 -m http.server 8000
```

## Deploy to GitHub Pages
1. Push this folder to a repo (e.g. `Portfolio`).
2. Repo Settings → Pages → Deploy from branch → `main` / root.
3. Site will be live at `https://<username>.github.io/<repo>/`.
